**AI Website Generator**
**An AI-powered full-stack web application that allows users to generate live websites from natural language prompts. Users can authenticate with Firebase, describe the website they want, and instantly preview AI-generated HTML, CSS, and JavaScript inside a secure sandboxed iframe.

**Features**
1 Firebase Authentication
Email & Password login
Google OAuth login
**AI Website Generation**
Users enter natural language prompts
Backend sends structured requests to the OpenAI API
AI returns complete HTML, CSS, and JavaScript
